Квадратные уравнения: теория и практика#

Введение#

Квадратным уравнением называется уравнение вида $ ax^2 + bx + c = 0 $, где $ a \neq 0 $.

Формула корней#

Корни вычисляются по формуле:

$$ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} $$

Выражение $ D = b^2 - 4ac $ называется дискриминантом.

Пример решения уравнения#

Решим уравнение $ 2x^2 - 4x - 6 = 0 $:

$ D = (-4)^2 - 4 \cdot 2 \cdot (-6) = 16 + 48 = 64 $

$ x = \frac{4 \pm \sqrt{64}}{4} = \frac{4 \pm 8}{4} $

$ x_1 = 3, \quad x_2 = -1 $

Онлайн-инструменты для построения графиков#

  • Desmos — очень удобный конструктор графиков. Вы вводите формулу, например, y = x^2, настраиваете внешний вид, делаете скриншот или экспортируете изображение (через меню «Поделиться» → «Изображение»).
  • GeoGebra — аналогичный функционал, можно сохранить график как PNG.
  • Wolfram Alpha — введите plot x^2, получите график, который можно скачать.

Преимущество: вы получаете уникальное изображение, точно соответствующее вашему заданию.

Алгоритм решения квадратного уравнения#

Примеры#

а) Решим уравнение: $4x^2 + 7x + 3 = 0$

$a = 4, b = 7, c = 3$

$ D = b^2 - 4ac = 7^2 - 4 \cdot 4 \cdot 3 = 49 - 48 = 1 > 0 $

два разных корня:

$ x_1 = -1,\quad x_2 = -0{,}75 $

б) Решим уравнение: $4x^2 - 4x + 1 = 0$

$a = 4, b = -4, c = 1$

$ D = (-4)^2 - 4 \cdot 4 \cdot 1 = 16 - 16 = 0 $

один корень:

$ x = 0{,}5 $

в) Решим уравнение: $2x^2 + 3x + 4 = 0$

$a = 2, b = 3, c = 4$

$ D = 3^2 - 4 \cdot 2 \cdot 4 = 9 - 32 = -13 < 0 $

данное уравнение корней не имеет.

Решите самостоятельно#

$ 2x^2 - 5x + 2 = 0 $

$ x^2 - 8x - 84 = 0 $

$ 0,8x^2 - 4x + 5 = 0 $

$ 7x^2 - 5x - 6 = 0 $

$ 9x^2 - 12x + 4 = 0 $

$ 2x^2 + x - 3 = 0 $

$ 3x^2 + 2x + 5 = 0 $
