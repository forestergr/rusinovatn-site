Структура урока-практикума (модульная технология)#

Для проведения уроков в режиме модульной технологии использую данную структуру урока-практикума (рекомендованную на курсах повышения квалификации).

№ Этапы урока Деятельность учителя Деятельность ученика
1. Оргмомент. Проверка готовности рабочего места. Проверка готовности рабочего места.
2. Актуализация знаний. 1) Консультации по возникшим вопросам.
2) Предлагает выполнить диагностическую работу (если в результате проверки домашней работы выяснилось, что у кого-то более половины работы выполнено неверно, им сразу можно предложить задания для работы на I уровне).		 1) Рефлексия по домашней работе (после самопроверки домашней работы либо по образцам, выполненным на доске, либо в результате взаимопроверки тетрадей).
2) Выполняют диагностическую работу.
3. Мотивация, целеполагание. Предлагает выполнить самопроверку (либо по готовым ответам, либо взаимопроверку), предлагает выбрать уровень для дальнейшей работы, фиксирует его в листе контроля, в случае необходимости помогает сформулировать учебные задачи. Выполняют самопроверку самостоятельной работы, выбирают уровень для дальнейшей работы, формулируют учебные задачи.
4. Практическая работа. Предлагает разноуровневую систему заданий, оказывает помощь в случае необходимости. Выполняют практическую работу в соответствии с выбранным уровнем, осуществляют самоконтроль либо обращаются за помощью учителя (в случае необходимости).
5. Подведение итогов урока, рефлексия. Предлагает выполнить соответствующие пункты рабочего листа. Дети проводят самоанализ деятельности на уроке.
6. Домашнее задание. Предлагает уровневое домашнее задание. Выбирает уровень домашнего задания, записывают в тетрадь упражнения или номера из учебника.
