Производная, применение производной

Производная, применение производной Цели урока: Образовательная: создать условия для самодиагностики и самокоррекции имеющихся знаний и умений применять их в стандартных и нестандартных ситуациях. Развивающая: создать условия для развития аналитических навыков. Воспитательная: создание условий для адекватной самооценки, воспитание воли и упорства в достижении поставленной цели. Тип урока: урок комплексного применения ЗУН учащихся. Структура урока-практикума может выглядеть так: Этап урока Деятельность учителя Деятельность ученика Оргмомент. Проверка готовности рабочего места. Проверка готовности рабочего места. Актуализация знаний. Консультации по возникшим вопросам. Предлагает выполнить диагностическую работу(если в результате проверка домашней работы выяснилось, что у кого-то более половины работы выполнено неверно, им сразу можно предложить задания для работы на I уровне). Рефлексия по домашней работе (после самопроверки домашней работы либо по образцам выполненным на доске, либо в результате взаимопроверки тетрадей). Выполняют диагностическую работу. (Приложение 1) (Приложение 2) Мотивация, целеполагание. Предлагает выполнить самопроверку (либо по готовым ответам, либо взаимопроверку), предлагает выбрать уровень для дальнейшей работы, фиксирует его в листе контроля, в случае необходимости помогает сформулировать учебные задачи. Выполняют самопроверку самостоятельной работы, выбирают уровень для дальнейшей работы, формулируют учебные задачи. (Приложение 3) Практическая работа. Выполняют практическую работу в соответствии с выбранным уровнем, осуществляют самоконтроль, либо обращаются за помощью учителя (в случае необходимости). (Приложение 4) Подведение итогов урока, рефлексия. Предлагает выполнить соответствующие пункты рабочего листа. Дети проводят самоанализ деятельности на уроке. (Приложение 5). Домашнее задание.
